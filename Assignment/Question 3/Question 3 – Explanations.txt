4. Inode Verification
The inode numbers of sample_data.txt, sample_hard.txt, and sample_soft.txt were displayed using the ls -i command to compare how Linux tracks files.

5. Inode Analysis
sample_data.txt and sample_hard.txt share the same inode number because hard links reference the same inode and data blocks.
sample_soft.txt has a different inode because it is a separate file that stores only the path to the original file.
